Puzzlehunt tools

List of useful tools for puzzlehunts. Please let me know if you think there's more I could add here! :menherateehee:

WOF tools

WOF stands for "Wheel of fortune". It means trying to guess the answer when you only know a few letters. Very useful for cheesing, or solving crossword clues.

  • Nutrimatic - for searching words and phrases using regex and anagrams
  • Onelook - for searching synonyms and related words
  • Qat - for searching words with very complex patterns
  • Util.in - for trigram hell

    • Cipher tools

    Most of the time dCode already has everything you need for puzzle ciphers.

  • dCode - has pretty much a tool for every cipher you'll ever need
  • tools.qhex.org - bruteforces every possible cipher or extraction method
  • CyperChef - lets you chain together multiple ciphers and more. useful if you don't want to code
  • Quipquip - for solving cryptograms

  • Braille decoder - to quickly decode braille. accepts wildcards too
  • Semaphore decoder - to quickly decode semaphore. accepts wildcards too
  • Binary decoder - to quickly decode binary. accepts wildcards too
  • Base64 decoder - to decode Base64 with wildcards.
  • Unicode lister - to get the unicode names of a string

  • Barcode Reader - for scanning barcodes, QR codes, etc
  • Puzzlehunt copy-pasteables - copy-pasteable Python snippets of various ciphers

    • Logic puzzle tools

    For solving logic puzzles.

  • Comparison of logic puzzle solvers - list of autosolvers for tons of puzzle genres
  • Puzz.link - list of tons of puzzle genres and editing tools
  • Penpa+ - for editing logic puzzles in general

  • dCode Word Search Solver - for wordsearch puzzles

    • Crossword tools

    Most of the time you can already solve most crossword clues by copy-pasting "[clue] cw clue" to Google.

  • ChatGPT - an AI that can quickly solve a bunch of crossword clues simultaneously
  • Wordplays.com - for searching crossword clue answers, if Google and ChatGPT is not enough
  • Crossword Genius - an AI that can solve cryptic clues. only available on iOS though

    • Audio search tools

    Searching audio is hard. :pomicry:

  • Aha Music - for uploading audio to search. it only allows up to 10 files a day, but I can't find any other alternative

    • Spreadsheet tools

    Useful tools when working with Google sheets.

  • Google Sheets Puzzle Tricks - contains lots of helpful sheets formulas and tips
  • Yet Another Puzzlehunt Spreadsheet Tutorial - another collection of helpful sheets formulas and tips
  • Crossword Parser - for converting crossword images to Google sheets

    • Python libraries

    If you love cheesing with Python.

  • solvertools - library for WOF or extraction
  • puzzle-tools - library for ciphers and datasets
  • grilops - grid logic puzzle solver library

    • Chinese puzzlehunt tools

    Chinese puzzlehunts like CCBC require additional tools because, well, they're in Chinese.

  • Google Translate - for translating Chinese text, and images too
  • Handwritten Chinese input - find a Chinese character by drawing it
  • 千篇国学 - for searching Chinese idioms
  • Zi.tools - to find Chinese characters based on parts of the character

    • Misc

  • Sets of Things - list of sets of things
  • Puzzlehunt Datasets - more list of sets of things

  • Puzzlehunt cheatsheet - puzzlehunt ciphers condensed in a single image (credit: @CALS)
  • Symbol ciphers reference list - list of various symbol ciphers

  • Have you tried... - list of things to try when stuck
  • betaveros's Have you tried? - more list of things to try when stuck
  • MIT Puzzle Tools - list of more puzzlehunt tools

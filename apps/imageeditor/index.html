JavaScript Image Editor

To create a new image, you will write a function called setpixel(x,y). It should return an array of length 4, [r,g,b,a], containing the RGBA values of pixel (x,y) in the new image.

Parameters:

x → x-coordinate of the pixel
y → y-coordinate of the pixel

Available functions:

pixel(x,y) → returns the RGBA value of the pixel at position (x,y) as an array [r,g,b,a]

Extra variables:

r → R value of pixel(x,y)
g → G value of pixel(x,y)
b → B value of pixel(x,y)
a → A value of pixel(x,y)

w → image width
h → image height

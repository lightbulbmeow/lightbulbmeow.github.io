Unicrush (recreation)

This is my recreation of Cary and Michael Huang's Unicrush, just for fun. The original is missing a few transformations, such as [erg → ㋍].

In this version, I only use digraphs and character decomposition mappings explicitly defined in Unicode standard. It seems more "complete" in that it even includes Japanese and Arabic character combinations.

However, even that is still incomplete. It doesn't include some transformations like [Pts → ₧].
